Jira Dashboards provide a **customizable interface** to display key project information using various gadgets. Dashboards help teams visualize data and track project progress at a glance.

### Commonly Used Gadgets

- **Issue Statistics**
  - Displays statistics based on issue types, priorities, or custom fields.
- **Sprint Burndown**
  - Visualizes sprint progress by showing remaining work over time.
- **Pie Charts**
  - Shows issue distribution by status, assignee, priority, or other fields.
